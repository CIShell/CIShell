Visualizations Tutorial
Author: Bruce Herr
Last Modified: April 20, 2005

Summary

Visualizations provide a visual representation of a data model. Visualization Plug-ins are created in the same way that Algorithms are with only a couple extra steps.

Pre-requisites

You will need to be familiar with Algorithm creation and the GUI Toolkit you intend to use.

Visualization Creation

In order to create a Visualization Plug-in, you will first need to create an Algorithm Plug-in using the steps described here. Usually you can delete the Algorithm class, unless you would like to use it to encapsulate your visualization, but usually you can just instantiate your visualization from the launch() method from the Plugin class.

If you wish to use the SwtCompositeBuilder or other classes from the edu.iu.iv.gui.* packages you must open up your plugin.xml, click the Dependencies tab, click the Add button, choose the edu.iu.iv.gui Plug-in (this should be available if you have the IVC Plug-in Development Plug-in installed in Eclipse), and save your plugin.xml

The actual visualization you create can be done in Swing, SWT, or whichever toolkit you would like. You just need to instantiate and show this class whenever the launch method is invoked.
