Sampledata Tutorial
Author: Bruce Herr
Last Modified: September 8, 2005

Summary

This tutorial shows how to make a Plug-in to provide sample data that may be used in the IVC.

Pre-requisites

You will need to have a Plug-in in mind that you wish to provide sample data for and have sample data to put in the sampledata folder.

Creating a Sampledata Plug-in

Providing sample data is a simple process thanks to our IVC Plug-in templates.

To create a sampledata plug-in project, go to File->New->Project... and select Plug-in Project from the Plug-in Development section. Make sure to give the project a name.

NOTE: Be sure to leave the "Create a Java project" checkbox checked, because there is some code that is created that needs to be run to put sampledata in the correct spot when starting the IVC. Also, leave the "Create an OSGI bundle manifest for the plug-in" unchecked.

Fig1: The plug-in project dialog should look like this.

Press Next > and fill out the plug-in information on the next page of the dialog.

WARNING: Be sure to uncheck the "Generate the java class that controls the plug-in's life cycle (recommended)" checkbox or your Plug-in will not work!
Also, in newer versions of Eclipse, the Runtime Library is worded Classpath instead. For this plug-in you will want to provide a jar name for it to work correctly.

Fig2: Enter in the desired information and press Next >.

Press Next > and check the "Create a plug-in using one of the templates". Choose IVC Sampledata template from the list of options.

Fig3: Select the desired template and press Next >.

Press Next > and fill in the details for your sampledata plug-in. The package name and sampledata class name is only for the loader class that will put your files in the right spot in the IVC. You will need to know the plug-in id of the plug-in you wish to provide sampledata for if you are publishing sampledata for a specific plugin. "The Name of the subdirectory..." textbox specifies what subdirectory to put the data in when the IVC is launched, which is usually under 'ivc-base-directory'/sampledata/'the subdirectory you choose'.

Fig4: Enter in the desired information and press Next >.

Press Finish and the template will create the project and will show up in your Eclipse workspace. You may then place the sample data you wish to provide in the top level sampledata folder of the project.
